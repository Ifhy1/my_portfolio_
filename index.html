☁ Serverless · Cloud Resume Challenge
Dynamic Cloud Resume v2
Evolution of a static resume site with a fully serverless backend. Features a real-time visitor counter — Lambda increments a DynamoDB count on each visit, surfaced via API Gateway, with the frontend hosted on S3 and served globally through CloudFront.
AWS S3CloudFrontLambda (Python)DynamoDBAPI Gateway

⚡ Serverless · AWS Lambda
Serverless URL Shortener
A fully serverless URL shortener deployed on AWS. Users submit a long URL and receive a short link that redirects to the original. Built with two Lambda functions, API Gateway routing, and DynamoDB for storage with automatic link expiry via TTL. Infrastructure defined entirely as code using AWS SAM.
AWS LambdaAPI GatewayDynamoDBAWS SAMNode.jsIAMus-east-1

🌐 Cloud Hosting · Static Site
AWS Cloud-Native Static Website
A secure, globally distributed static website hosted on S3 and served via CloudFront. Configured Origin Access Control (OAC) to keep the S3 bucket fully private, applied a least-privilege bucket policy, and leveraged AWS edge caching to minimise latency worldwide.
